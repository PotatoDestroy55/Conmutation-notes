Comandos para configurar switches en vlan

Quieres ver como configurar inter-VLAN's, checa este articulo

Comando para crear VLANS

  1. Entramos a la configuración de terminal. 

            config t

  2. Introducimos a la sección VLAN junto al número que deseamos crear 

            vlan *vlan id*

  3. Le colocamos un nombre a la VLAN, esto para identificarlo nosotros. 

            name *nombre vlan*

Comandos de asignación de puertos VLAN

  1. Entramos a la configuración de terminal. 

            config t

  2. Nos colocamos en las interfaces que queremos asignar a la vlan

    1. Si se hace con un rango se coloca de la siguiente manera 

      interface range *rango de interaz*  

interface range f0/1 - 16

    2. Si se quiere hacer con una interfaz se coloca la única interfaz que se quiera 

      interface *valor de la interfaz*

interface f0/1

    3. Si son interfaces separadas se puede utilizar comas. 

      interface *interfaz1*, *interfaz2*, ...  

interface f0/1, f0/15, f0/24

  3. Ahora dentro de las interfaces se establece el modo acceso. 

            switchport mode access

  4. Se Asigna el puerto a una VLAN. 

            switchport access vlan *vlan-id*

  5. Se vuelve al modo EXEC privilegiado

            end

Mostrar la tabla de VLANS en un switch

  1. Entramos al modo EXEC privilegiado. 

            enable

  2. Colocamos el siguiente comando para que nos muestre la tabla VLAN 

            show vlan brief

Mostrar el valor de una interfaz en el switchport

  1. Entramos al modo EXEC privilegiado. 

            enable

  2. Colocamos el siguiente comando para que nos muestra la información del puerto requerido 

            show interfaces *número del puerto* switchport  

        Ejemplo  
        show interfaces G0/0 switchport

Eliminar las VLAN

  1. Entramos a la configuración de terminal. 

            config t

  2. En la configuración global del switch escribimos el siguiente comando para quitar una VLAN

            no vlan *vlan id*

Precaución: Si se elimina una VLAN con puertos asignados estos no se podrán comunicar con los hosts que esten conectados en dichos puertos, por lo que siempre se deben asignar con anterioridad a otra VLAN antes de eliminarla.

Comandos de configuración troncal

  1. Entramos a la configuración de terminal. 

            config t

  2. Nos colocamos en las interfaces que queremos asignar el troncal

    1. Si se hace con un rango se coloca de la siguiente manera 

      interface range *rango de interaz*  

interface range f0/1 - 16

    2. Si se quiere hacer con una interfaz se coloca la única interfaz que se quiera 

      interface *valor de la interfaz*

interface f0/1

    3. Si son interfaces separadas se puede utilizar comas. 

      interface *interfaz1*, *interfaz2*, ...  

interface f0/1, f0/15, f0/24

  3. Una vez seleccionado los puertos, establecer el puerto(s) en modo de enlace troncal permanente. 

            switchport mode trunk

  4. Cambiar la configuración de la VLAN nativa a otra opción que no sea VLAN 1. 

            switchport trunk native vlan *vlan-id*

  5. Especificar la lista de VLAN que se permitirán en el enlace troncal. 

            switchport trunk allowed vlan *vlan-list*

  6. Volver al modo EXEC privilegiado.

            end

Configurar la ip en un switch en una vlan

  1. Acceder a la interfaz de VLAN deseada 

            interface vlan *vlan id*

  2. Colocar la ip a la vlan 

            ip addres *ip switch* *mascara de subred*

  3. Activar el puerto 

            no shutdown

Colocar la default gateway a un switch

  1. Entramos a la configuración de terminal. 

            config t

  2. Introducimos el IP que va a fungir como default gateway 

            ip default gateway *ip de puerta de enlace predeterminada*
